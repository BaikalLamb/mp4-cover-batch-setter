# mp4-cover-batch-setter
# MP4 批量封面设置工具

一个简单的 Windows 图形化工具，用于批量给 MP4 视频写入同一张内嵌封面图。

适合连续剧集、课程视频、合集视频等场景：选择一张 JPG/PNG 图片，再选择视频文件夹，即可批量处理多个 MP4 文件。

## 功能特点

- 批量为 MP4 写入统一封面
- 支持 JPG、JPEG、PNG 封面图片
- 视频轨、音频轨使用复制模式，不重新压缩
- 支持输出到新文件夹
- 支持覆盖原文件，覆盖前会先生成临时文件
- 支持包含子文件夹
- 内置 FFmpeg/FFprobe，无需用户额外安装环境

## 使用方法

1. 下载 Release 里的 `MP4批量封面设置工具.exe`
2. 打开程序
3. 选择封面图片
4. 选择 MP4 视频所在文件夹
5. 选择输出文件夹，或勾选覆盖原文件
6. 点击“开始批量设置封面”

## 重要说明

本工具不会重新压缩原视频和音频，处理时主要修改 MP4 容器中的封面相关数据。

但需要注意：

- 不同播放器、系统资源管理器对 MP4 内嵌封面的支持不完全一致
- Windows 文件资源管理器可能受到缩略图缓存影响，不一定马上显示新封面
- 如果某些播放器不支持 MP4 内嵌封面轨，可能仍会显示视频第一帧
- 如果要求所有平台都强制显示指定图片，通常需要把图片真正做成视频第一帧，这会修改视频内容

## 技术原理

程序基于 Python Tkinter 编写界面，并调用 FFmpeg/FFprobe 完成处理。

处理逻辑大致为：

- 读取 MP4 流信息
- 排除旧的内嵌封面轨
- 保留原视频、音频、字幕等普通轨道
- 将选择的封面图片转换为兼容性较好的 JPEG 封面
- 使用 `-c copy` 复制原始媒体流
- 将封面作为 `attached_pic` 写入 MP4

核心思想是：尽量只改封装和元数据，不改视频画质。
